# Student Academic Performance Analysis

## Project Overview
Analyzed academic performance of 100+ students across 4 subjects 
to identify at-risk students, subject-wise trends, and performance 
gaps using Python and Pandas.

## Business Problem
An educational institution needed to identify underperforming students 
early, understand which subjects had the lowest scores, and propose 
targeted intervention strategies to improve results.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## Key Steps
1. Created and cleaned dataset with 100+ student records across 4 subjects
2. Performed Exploratory Data Analysis (EDA)
3. Calculated subject-wise averages and total scores
4. Identified top scorers and at-risk students (bottom 20%)
5. Built grouped bar charts and pie charts for visual reporting
6. Proposed intervention strategy based on data insights

## Key Business Insights
- English and Tamil had higher average scores than Maths and Science
- Bottom 20% of students identified for targeted academic support
- Top scorers averaged 88.75 marks across all subjects
- Subject-wise gap analysis revealed Maths needed most improvement

## KPIs Tracked
- Subject-wise Average Score
- Student Pass/Fail Rate
- Top Performer Identification
- At-Risk Student Cohort (Bottom 20%)
- Score Distribution by Subject
