# Sales Data Analysis

## What This Project Does
This project analyzes 6 months of sales data across different products using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib

## Steps I Followed
1. Created a dataset with Month, Product, Sales, and Revenue details
2. Calculated total sales, total revenue, best selling product, and highest revenue month
3. Built a bar chart to show total sales by product
4. Built a line chart to show monthly revenue trend

## What I Found
- Laptop is the best selling
